= Viewing Salesforce Account and Portal Information
ifndef::env-site,env-github[]
include::_attributes.adoc[]
endif::[]

The *Settings* page provides information about the Salesforce account and API portal. 

To view information about the Salesforce organization and the portal, from the *Manage your API portal* page, click *Settings*. 


Salesforce Org information:

* Environment
* Username
* Sandbox name
* Organization domain
* Organization ID

API Portal information:

* Status
* Publish status
* API portal URL
