Anypoint API Community Manager licenses are procured and provisioned by MuleSoft. To set up an organization, you can have the following Salesforce organizations:

* Unlimited Edition
* Enterprise Edition
* Performance Editions of Salesforce

API Experience Hub has five administrator licenses. You need one only to perform the Salesforce linking process.
