The API Details page includes the following information about an API that the portal administrator sets up. 

* Name
* Type
* Description
* Contact email
* Version
* Client applications

The API Console is located in the lower section of the API Details page. The console component is interactive and shows information about REST APIs and their methods. Endpoints are displayed in alphabetical order.
See xref:api-console-specification-and-documentation.adoc[API Console Specification and Documentation]for more information.

To view the API Details page, select an API from Home or APIs.
